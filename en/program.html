Core chapters

1. Mathematical Modeling

Wolfram Mathematica

Mathematical notation, special character aliases, converting formuli to and from LaTeX format. Everything is an expression: operators, functions, data types. Symbolic computation principles. Patterns and transformation rules, assigning variables and functions. Functional programming, listable functions, functional operations. Localized transformation rules. Solving algebraic and differential equations. Plots, graphics objects, interactive outputs. Animation, spline animation, and exporting frame sequences.

2. Raster Graphics

Adobe Phostoshop

General interface and document structure, shortcuts for navigation and layer manipulation directly from the document window. Navigator, History, Layers, and Tools panels. Drawing basics: Brush and Eraser tools, Fill options, plotting arrows. Changing image and canvas size, Free Transform mode. Selecting image regions, Marquee and Polygonal Lasso tools. Layer alignment and distribution. Nondestructive editing: Masks, Vector masks, and Clipping masks, Smart objects, Paths. Utilizing Channels of RGB image. Changing brightness: Curves and Levels functions, Histogram, Dodge and Burn tools, Adjustment Layers. Retouching: Stamp and Healing Brush tools, Content-Aware Fill. Nonlinear history, History Brush tool. Automation: Actions panel, Batch and Variables functions, Scripting.

3. Vector Graphics

Adobe Illustrator

Mathematical foundations, Bézier curves. General interface and document structure: similarities and differences with Adobe Photoshop, distinctions in the Layers panel, Isolation mode. Making and editing splines: Pen tool and its controls, adding spline points, Anchor Point tool, joining splines. Compound paths: Pathfinder panel, Shape Builder tool, Clipping Masks. Appearance panel, dashing and arrowheads, Expand Appearance, selecting objects by properties and types. Text layers, text block threading, typing along splines, convertion to outlines. Embeding external files, combining vector and raster images, linking between Illustrator and Photoshop, tracing raster images. Importing LaTeX documents, dealing with font issues (font sources from CTAN, selectively replacing missing fonts, Flatten Transparency), generating equations in-place with LaTeX2AI. Vector image formats, exporting as raster image.

Advanced chapters (alternative choice)

4A. 3D Graphics and Animation

Unreal Engine

General interface: Viewport navigation and basic tools, Outliner, Details, and Content Drawer panels. Setup for low-end PC. Adding basic shapes and assembling a scene: left-handed coordinate system!, object parenting. Keyframe animation, Sequencer panel, Movie Render Queue plugin, render settings for high-quality video. Actors, programming with Blueprints, real-time physics simulations. Importing keyframe animations, table data, and 3D meshes from Wolfram Mathematica. Importing LaTeX expressions via Adobe Illustrator, Widgets, 2D and 3D rendering of text and imported formuli. Trajectory visualisation with Niagara. Basic materials, Material Blueprints, and Material Instances. Lighting and camera settings, Post Process Volume, HDRI.

4B. Generative Neural Networks

Stable Diffusion

Параметризация функционального пространства, универсальная теорема аппроксимации. Принципы обучения нейросетей для классификации и обработки изображений. Автокодировщики и их применение для анализа динамических систем. Конволюционные нейросети (CNN), нейросети для улучшения качества изображений и видео. Нейросети для генерации изображений: вариационные автокодировщики (VAE), генеративно-состязательные сети (GAN), дифузионные модели. Работа со Stable Diffusion в Google Colab. img2img

4C. Web and Interactive design

HTML + CSS + JavaScript

HTML basics: tags, attributes and styles. Каскадные таблицы стилей (CSS): селекторы и специфичность, наследование, комбинаторы, псевдоклассы, переменные. Размеры элементов, абсолютные и относительные единицы измерения, разница между padding, margin, border и outline. Способы задания положения элементов: свойства position и display, макеты flexbox и grid, сокращённое свойство inset. LaTeX-формулы с MathJax. CSS переходы и анимации по ключевым кадрам. SVG в HTML, задание SVG атрибутов через CSS. Объектная модель документа (DOM), изменение свойств объектов при помощи JavaScript. Базовые интерактивные элементы, атрибут onclick, обработка событий с addEventListener, использование setTimeout и setInterval.

Extra chapters (on demand)

1*. 2D Animation and Video editing

Adobe After Effects

Интерфейс программы, панели Project, Composition, Timeline и Preview. Импорт видео, покадровой анимации, psd и ai файлов. Растровые, векторные и текстовые слои. Свойства слоёв, группа свойств Transform, опция Parent. Ключевые кадры, временная интерполяция. Функция Graph Editor, редактирование графиков инструментом Pen. Выражения. Опции Motion blur и Frame Blending. Настройки экспорта видео, панель Render Queue. Базовая 3D-графика в After Effects. Задание трёхмерного вращения объектов, системы отсчёта и сложное движение. Камеры и источники света. Плагин Element 3D, создание 3D-моделей из векторных и текстовых слоёв, импорт моделей из внешних источников, настройки материалов. Методы для преобразования координат и группа эффектов Expression Controls. Импорт данных из Wolfram Mathematica.

2*. Presentation and Layout design

Adobe InDesign | Microsoft PowerPoint

Базовые принципы дизайна. Шрифты: классификация, форматы файлов, инструменты для подбора и поиска шрифтов. Использование цвета в дизайне. Верстка постеров в Adobe InDesign. Оформление презентаций в Microsoft PowerPoint, интерактивные элементы, анимация при помощи встроенных инструментов.

3*. Color Theory

Constructing RGB color space from the ground up. Curvilinear color coordinates: HSL, HSV, and LAB. Differences between Hue, Shade, Tone, and Tint. Conversion of images to grayscale and CMYK. Color (bit) depth, HDRI. Blending modes in Adobe Photoshop and Wolfram Mathematica, multiply as a substitute for masking. Hue/Saturation adjustment and selective color editing. Colorizing grayscale images, working with channels, gradient maps in Wolfram Mathematica, Adobe Photoshop, and After Effects. Color decomposition for retouching.

4*. Procedural Image Processing and Generation

Image convolution, linear and box kernels, separation of variables. Motion (directional) blur, camera blur, Gaussian blur. Drop shadows and glow. Frequency decomposition in image processing, wavelets. Sharpening filters: Sharpen, High Pass, Unsharp Mask. Edge detection. Image deconvolution and denoising, loss-of-information obstacles (out-of-bounds blur, noise infinite frequency). Random number generation (LCG and Mersenne Twister from the ground up, relation to logistic map and Lorenz attractor), random seed. Gaussian and fractal noise, implementation in Adobe Photoshop and After Effects. Reaction–diffusion.

5*. 3D Programming Fundamentals

GLSL | Wolfram Mathematica

Mathematical foundations: ray tracing vs projection (rasterization) rendering. Ray tracing variations: path tracing, ray marching, cone marching. Physics-based rendering (PBR) materials: the rendering equation, Biderectional Scattering Distrubution Function (BSDF), metallic-roughness model. Specular reflections: Fresnel coefficients, Schlick's approximation, highlight models. Geometry and normals: Phong shading, normal maps. Global illumination and ambient occlusion, implementation with Fibonacci lattice. Anti-aliasing, motion blur, and depth-of-field blur with combined supersampling technique. Shaders: real-time GPU simulations and rendering, integration into HTML with <canvas>, Unreal Engine material blueprints. Native 3D functionality in Wolfram Mathematica: materials and lighting, generating and exporting 3D meshes.
